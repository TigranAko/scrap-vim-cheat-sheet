
# Парсер шпаргалок Vim

Python-скрипт для парсинга шпаргалок по Vim с сайта [vim.rtorr.com](https://vim.rtorr.com) и сохранения данных в различных форматах (JSON, CSV) для разных языков.

## Возможности

- Парсит шпаргалки Vim для различных языков
- Извлекает комбинации клавиш и их описания
- Сохраняет данные в форматах JSON и CSV
- Организует вывод по языкам и категориям
- Обрабатывает специальное форматирование и разделы с советами

## Структура проекта

```

data/
├──htmls/                 # Исходные HTML-файлы
├──all_data/              # Полные JSON-данные для каждого языка
└──{язык}/               # Директории для каждого языка с категоризированными данными
├── {категория}.json
└── {категория}.csv

```

## Требования

- Python 3.x
- requests
- beautifulsoup4
- lxml

## Установка

1. Клонируйте репозиторий или скачайте скрипт
2. Создайте виртуальное окружение:

```bash
# Создание виртуального окружения
python -m venv venv

# Активация на Windows
venv\Scripts\activate

# Активация на Linux/Mac
source venv/bin/activate
```

## Установите необходимые пакеты:

```bash
pip install requests beautifulsoup4 lxml
```

или 

```bash
pip install -r requirements.txt
```

## Запустите основной скрипт:

```bash
python main.py
```

## Скрипт выполнит следующие действия:

1. Создаст необходимые директории
2. Скачает главную индексную страницу
3. Извлечет доступные ссылки на языки
4. Скачает шпаргалку для каждого языка
5. Распарсит и извлечет данные в структурированные форматы
6. Сохранит данные в форматах JSON и CSV

## Деактивация виртуального окружения

После завершения работы с проектом деактивируйте виртуальное окружение:

```bash
deactivate
```

## Формат выходных данных

### Структура JSON

```json
{
  "название_категории": {
    "item_1": {
      "keys": "комбинация vim",
      "description": "описание действия"
    },
    "item_2": {
      "keys": "другая комбинация", 
      "description": "другое описание"
    },
    "wells": ["полезный совет 1", "полезный совет 2"]
  }
}
```

### Структура CSV

Каждая категория генерирует CSV-файл со столбцами:

· keys: Комбинация клавиш Vim
· description: Описание действия комбинации

## Источники данных

Данные парсятся с https://vim.rtorr.com/ - комплексного ресурса со шпаргалками по Vim, включая языко-специфичные комбинации и советы.
